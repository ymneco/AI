# 工場3Dスキャン 撮影ガイド

## 概要

このガイドは、工場内の間取り・機械配置を3Dモデルとして再構成するための動画撮影手順を説明します。
一般的なスマートフォン（iPhone/Android）で撮影可能です。

---

## 撮影前の準備

### 必要機材
- スマートフォン（2020年以降のモデル推奨）
- 十分なストレージ空き容量（1エリアあたり500MB〜1GB）

### スマートフォンの設定
- **解像度**: 1080p（FHD）以上。4Kだとより良い結果が得られますが、ファイルサイズが大きくなります
- **手ぶれ補正**: ONにしてください
- **フレームレート**: 24fps〜30fps（デフォルトでOK）
- **フラッシュ/ライト**: OFF（自然光・室内照明を使用）

### スケール基準物（推奨）
精度の高い寸法計測が必要な場合は、以下のいずれかを撮影エリアの床に置いてください:
- A4用紙（210mm x 297mm）
- A3用紙（297mm x 420mm）
- メジャーテープ（1m）

---

## 撮影手順

### 基本ルール

1. **ゆっくり歩く** — 通常の歩行速度の半分（秒速0.5m程度）
2. **カメラを水平に保つ** — 上下の傾きを最小限に
3. **急な回転を避ける** — 方向転換は3〜5秒かけてゆっくり
4. **同じ場所を複数角度から** — オーバーラップ（重複）が精度の鍵

### 撮影パターン

#### パターンA: 小〜中エリア（50m²以下）

```
エリアの外周を一周し、次に中央を通る「8の字」パターン

    ┌───────────────────┐
    │  ──→──→──→──→──   │
    │ ↑               ↓ │
    │ │   ←←←←←←←←←  │ │
    │ │  │           ↓  │
    │ │  │  →→→→→→→ │  │
    │ ↑  ↓           ↑  │
    │  ──←──←──←──←──   │
    └───────────────────┘
```

- 所要時間: 1〜2分
- 外周を歩きながらカメラで内部を撮影
- 中央部も通って死角をなくす

#### パターンB: 大エリア（50m²超）

エリアを25m²程度のゾーンに分割し、各ゾーンをパターンAで撮影。

**重要**: ゾーン間の**接続部分を5秒以上撮影**してください。
これにより、システムが各ゾーンを自動的に結合できます。

```
┌──────────┬──────────┐
│  Zone A  │  Zone B  │
│          ┆          │
│     接続部分(5秒+)   │
│          ┆          │
├──────────┼──────────┤
│  Zone C  │  Zone D  │
└──────────┴──────────┘
```

### 撮影時の注意事項

#### やるべきこと
- 壁や柱を含むように撮影する（構造物が3D復元の基準になる）
- 同じ物体を異なる角度から映す（最低3方向）
- 天井や床も適度に映す
- 明るい場所で撮影する

#### 避けるべきこと
- 急な方向転換やパン（3D復元が切れる原因に）
- カメラを地面や天井だけに向け続ける
- 手で撮影エリアを遮る
- 強い逆光の場所での撮影
- 動いている人や車両が多い場面（ノイズになる）

### 工場環境での注意

| 状況 | 対策 |
|------|------|
| 蛍光灯のちらつき | 気にしなくてOK（システムが対応） |
| 金属の反射面 | 複数角度から撮影する（正面から見ると反射で白飛びする） |
| 暗い場所 | 可能なら照明を追加。無理なら撮影速度をさらに遅く |
| 動いている人 | 可能なら人がいない時間帯に撮影 |
| ガラス面 | 透明な面は復元できないため、斜めから撮影する |

## 撮影後

### ファイルの送付
- 動画ファイル（MP4）をそのまま送付してください
- ファイル名に撮影日時とエリア名を入れてください
  - 例: `20260326_工場A_エリア1.mp4`
- 写真（JPG）の場合はフォルダにまとめてZIP圧縮

### 品質チェック
撮影した動画を見返して以下を確認:
- 全体を通してぶれが少ないか
- エリア全体がカバーされているか
- 明るさが極端に変わる箇所がないか

## よくある質問

**Q: 1本の動画は何分くらいが適切ですか？**
A: 1エリアあたり30秒〜2分程度。長すぎると処理時間が増えます。

**Q: 写真でも大丈夫ですか？**
A: はい。1エリアあたり15〜30枚の写真でも可能です。
同じ場所を複数角度から撮り、隣り合う写真の70%以上が重複するようにしてください。

**Q: 4Kと1080pどちらが良いですか？**
A: 4Kの方がディテールが良くなりますが、ファイルサイズが4倍になります。
バランスを取るなら1080pで十分です。

**Q: 縦向きと横向きどちらで撮影すべきですか？**
A: **横向き（ランドスケープ）推奨**。より広い範囲を1フレームでカバーできます。
